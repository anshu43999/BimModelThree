## 使用规则

1. 归档文档中的“当前状态”“排期”和接口示例默认视为历史信息。
2. 当前文档与归档文档冲突时，以根目录 `docs/README.md` 指定的权威文档为准。
3. 不直接修改归档文档的历史结论；需要重新启用时，在主目录创建或更新当前文档。
4. 归档只移动不删除，保留需求和决策追溯能力。
