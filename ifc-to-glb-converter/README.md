# ifc-to-glb-converter

Convert IFC building models to glTF Binary (.glb) format.

Uses IfcOpenShell (OpenCASCADE geometry engine) for high-quality triangulation,
trimesh for scene assembly, and pygltflib for BIM property injection.

## Prerequisites

- Python 3.11 or later
- IfcOpenShell (`pip install ifcopenshell`)

## Installation

```bash
cd ifc-to-glb-converter

# Create and activate a virtual environment
python -m venv .venv
.venv\Scripts\activate     # Windows
# source .venv/bin/activate  # macOS / Linux

# Install the package in editable mode
pip install -e .
```

## Usage

```bash
# Basic conversion (output defaults to output/<stem>.glb)
ifc-to-glb convert path/to/model.ifc

# Specify output path and enable colour
ifc-to-glb convert model.ifc --output result.glb --color

# Flat mode — no spatial hierarchy
ifc-to-glb convert model.ifc --no-hierarchy

# Geometry only — no hierarchy, no properties
ifc-to-glb convert model.ifc --no-hierarchy --no-properties

# Inspect an IFC file without converting
ifc-to-glb info model.ifc
```

During development you can also run directly:

```bash
python -m ifc_to_glb.cli convert model.ifc --color
```

## Output

The resulting `.glb` file contains:

- **Geometry**: Triangle meshes generated by OpenCASCADE via IfcOpenShell
- **Hierarchy**: Spatial structure (Project → Site → Building → Storey → Elements)
- **Properties**: BIM attributes (GlobalId, Name, Description, etc.) in `node.extras`
- **Materials**: Per-type vertex colours when `--color` is enabled

## Running Tests

```bash
pip install pytest
pytest tests/ -v
```

To run end-to-end tests with a real IFC file:

```bash
# Use a test file from the sibling converter-service
$env:TEST_IFC_PATH = "../converter-service/models/20210219Architecture.ifc"
pytest tests/test_converter.py -v
```

## Project Structure

```
ifc_to_glb/
├── cli.py          # Typer CLI entry point
├── converter.py    # Main conversion orchestration
├── geometry.py     # IfcOpenShell geometry iterator wrapper
├── materials.py    # IFC type → colour mapping
├── hierarchy.py    # Spatial tree builder
├── properties.py   # BIM property extraction
└── exporter.py     # trimesh scene assembly + GLB export
```
